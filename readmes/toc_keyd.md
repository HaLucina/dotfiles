ソース（`keyd/etc/keyd/default.conf`）に基づき、READMEに記載する際に分かりやすい形式でキーマップの設定をまとめました。

この設定の最大の特徴は、**CapsLockキーをナビゲーション層への切り替えスイッチ**として活用し、ホームポジションを崩さずにカーソル操作やファンクションキー入力を行えるようにしている点です。

---

## ⌨️ Keyd キーマップ設定

システム全体で適用されるキーマップ設定（`keyd`）の概要です。

### 1. 基本設定 (Main Layer)
通常の打鍵時の挙動を変更しています。

| キー | 入力される機能 | 備考 |
| :--- | :--- | :--- |
| **CapsLock** | **Navレイヤー (ホールド時)** | 押している間のみナビゲーション層に移行 |
| `;` | `:` (コロン) | セミコロン単体でコロンを入力 |
| `Shift` + `;` | `;` (セミコロン) | シフト併用でセミコロンを入力 |
| `Ctrl` + `[` | `Esc` | Vim等で多用するEscを割り当て |
| 矢印キー | `noop` (無効化) | Fnキーの誤爆防止のため標準の矢印を無効化 |

### 2. ナビゲーション (Nav Layer)
**CapsLockを押しながら**入力する操作です。Vimライクな移動が可能です。

| キー | 機能 | 備考 |
| :--- | :--- | :--- |
| `h` / `j` / `k` / `l` | **← / ↓ / ↑ / →** | Vimスタイルの矢印操作 |
| `u` / `d` | **PageUp / PageDown** | ページ単位の移動 |
| `i` / `a` | **Home / End** | 行頭・行末への移動 |
| `g` | **Ctrl + Home** | 文頭へ移動 |
| `Shift` + `g` | **Ctrl + End** | 文末へ移動 |
| `Backspace` | **Delete** | バックスペースをデリートに変更 |

### 3. ファンクションキー (Nav Layer)
**CapsLockを押しながら**数字キー等を入力することで、F1〜F12を呼び出せます。

*   `1` 〜 `9` → **F1 〜 F9**
*   `0` → **F10**
*   `-` (minus) → **F11**
*   `=` (equal) → **F12**
